# SVG → PDF 変換ツール

ブラウザだけで SVG を PDF に変換します。

## 含まれるファイル

- `index.html` — 本体
- `.nojekyll` — GitHub Pages 用の補助ファイル

## 仕様

- SVG ファイルをドラッグ&ドロップ、または選択で読み込み
- 複数 SVG をまとめて変換可能
- ページサイズは以下から選択可能
  - SVG と同じサイズ
  - A4 に合わせる
  - Letter に合わせる
- 変換はブラウザ内で実行
- PDF は各ファイルごとに保存

## 注意点

- 外部フォントを参照する SVG は、表示環境に依存することがあります
- 外部画像参照を含む SVG は、参照先の取得状況によって見た目が変わることがあります
- 本ツールは CDN から以下のライブラリを読み込みます
  - `jsPDF`
  - `svg2pdf.js`
