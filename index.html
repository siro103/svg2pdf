SVG → PDF

GitHub Pages でそのまま動く、ブラウザ完結の変換ツールです。SVG ファイルをドロップして PDF を生成します。変換は端末内で実行され、サーバー送信は行いません。

SVG をここにドロップ
またはクリックして選択。複数ファイルもまとめて変換できます。
SVG ファイルを選択してください。
プレビュー
ここに SVG のプレビューが表示されます。
100%
青い枠をドラッグして、PDFに残す範囲を調整できます。外側へ広げると余白を足せます。見づらいときは +/- でプレビュー倍率を変えられます。
補足:
・変換はブラウザ内で行います。
・テキストや図形は可能な限りベクターのまま維持します。
・外部フォントや外部画像参照を含む SVG は、環境によって見た目が変わる場合があります。
